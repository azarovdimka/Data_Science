# 🔢 Data Science Portfolio

> Здесь собрана коллекция проектов по анализу данных, машинному обучению, LLM и разработке AI-агентов, демонстрирующих опыт работы с данными, создания ML-моделей, разработки интеллектуальных систем и деплоя веб-приложений. Добро пожаловать в мое портфолио по Data Science! 

## 📁 Структура проектов

### 📈 Data Analysis
*Исследовательский анализ данных и визуализация*

- **[Недвижимость]**: Выявление и оценка факторов, в наибольшей степени влияющих на цену жилой недвижимости.  
- **[Cамокаты Go Fast]**: Поиск способов увеличения прибыльности сервиса: проверка гипотез, ответы на поставленные вопросы.
- **[Скоринг должников]**: Выявление и оценка факторов, больше всего влияющих на просрочку по кредиту в банке.
- **[Скоринговая модель по поиску недооценнных акций на рынке]**: Парсинг мультипликаторов, при наличии недооценнности по определенному мультипликатору раздаются баллы, суммируются, выводится топ недооценнных акций на рынке.

### 🤖 Machine Learning
*Модели машинного обучения для решения бизнес-задач*

- **[Рыночная стоимость авто]**: Задача регрессии: определние рыночной стоимости авто. Показатель RMSE на 40% лучше цели.
- **[Молочная ферма]**: Разработка нескольких моделей, чтобы определить лучшую для предсказания удоя молока коровы и вкуса молока коровы перед покупкой коровы;
- **[Интернет-магазин: прогнозирование оттока VIP‑клиентов]**: Разработка моделей для определения снижающейся покупательской активности и для повышения покупательской активности;
- **[HR]**: Задача классификации: разработка моделей для определения уовлетворенности сотрудников и предсказания их увольнения из компании.
- **[Скважины]**: Практика применения техники Bootstrap в машинном обучении при прогнозировании объемов запасов с целью минимизации ошибки расчетов.
  
### 🌍 Веб-приложения
*Модели машинного обучения в структуре веб-приложений*

- **[Heart Attack Risk Prediction]**: API для предсказания риска сердечного приступа на основе медицинских показателей пациентов с использованием машинного обучения (CatBoost). Улучшил метрику до максимальных значений.
**Recall:** 0.99. 
Приложение доступно онлайн: http://vds.spb.su:8001

- **[Оценка за экзамен]**: Веб-приложение для автоматической классификации ответов инностранных граждан на экзамене по русскому языку. **f1_score**: 0.76 
  Веб-приложение представлено на сайте http://vds.spb.su:8000


### 🤖 AI-агенты и Telegram-боты
*Телеграм-боты и AI-агенты*
- **Информационно-справочный бот**: RAG-система: обучаемый Telegram-бот для бортпроводников авиакомпании с базой знаний по процедурам и регламентам.

## 📊 Ключевые навыки

• **Анализ данных:** EDA, статистический анализ, визуализация;

• **Machine Learning:** классификация, регрессия, кластеризация;  

• **Feature Engineering:** отбор и создание признаков, устранение мультиколлинеарности, несбалансированности классов;

• **ML Pipeline & Optimization:** Pipeline, GridSearchCV, подбор гиперпараметров, кросс-валидация;

• **AI Development:** создание чат-ботов и автоматизированных систем;

• **Web deploy:** разворачивание веб-приложений с моделями.
