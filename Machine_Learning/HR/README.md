# HR-аналитика: Предсказание удовлетворенности и оттока сотрудников

## Описание проекта

HR-аналитики компании «Работа с заботой» помогают бизнесу оптимизировать управление персоналом. Собрав данные путем опроса сотрудников, были поставлены две ключевые задачи для машинного обучения.

## Поставленные задачи

### Задача 1: Предсказание уровня удовлетворенности сотрудника
- **Цель**: Создать модель для предсказания уровня удовлетворенности сотрудника работой в компании
- **Целевая переменная**: `job_satisfaction_rate` (от 0 до 1, где 0 — совершенно неудовлетворен, 1 — полностью удовлетворен)
- **Тип задачи**: Регрессия

### Задача 2: Предсказание увольнения сотрудника
- **Цель**: Создать модель для предсказания вероятности увольнения сотрудника
- **Важность**: Внезапные увольнения несут риски для компании, особенно если уходит важный сотрудник
- **Тип задачи**: Классификация

## Используемые признаки

- **id** — уникальный идентификатор сотрудника
- **dept** — отдел, в котором работает сотрудник
- **level** — уровень занимаемой должности
- **workload** — уровень загруженности сотрудника
- **employment_years** — длительность работы в компании (в годах)
- **last_year_promo** — было ли повышение за последний год
- **last_year_violations** — нарушал ли сотрудник трудовой договор за последний год
- **supervisor_evaluation** — оценка качества работы сотрудника руководителем
- **salary** — ежемесячная зарплата сотрудника

## Методология решения

### Предобработка данных
- Создан пайплайн для автоматической предобработки данных
- Обработка пропущенных значений (удаление строк с пропусками)
- Исправление опечаток в данных (например, "sinior" → "senior")
- Удаление дубликатов
- Выявление и удаление выбросов методом IQR (только для тренировочной выборки)
- Анализ неявных дубликатов

### Используемые инструменты
- **Библиотеки**: pandas, numpy, matplotlib, seaborn, scikit-learn
- **Модели**: RandomForestRegressor, RandomForestClassifier, LogisticRegression, SVC, DecisionTree, KNeighbors
- **Метрики**: SMAPE, ROC-AUC, Precision, Recall, F1-score
- **Валидация**: GridSearchCV, cross-validation
- **Предобработка**: StandardScaler, MinMaxScaler, RobustScaler, OneHotEncoder

## Полученные результаты

### Задача 1 (Регрессия - удовлетворенность)
- **Метрика SMAPE**: 12.44
- **Качество**: Отличное для бизнес-задач (ошибка менее 15%)

### Задача 2 (Классификация - отток)
- **Метрика ROC-AUC**: 0.92
- **Качество**: Превосходное (выше 0.9 считается отличным результатом)

## Почему это хорошие результаты

### SMAPE 12.44%
- Ошибка предсказания удовлетворенности составляет менее 13%
- Для HR-задач это высокая точность, позволяющая принимать обоснованные решения
- Модель может надежно выявлять сотрудников с низкой удовлетворенностью

### ROC-AUC 0.92
- Модель правильно классифицирует 92% случаев увольнения
- Высокая способность различать сотрудников, склонных к увольнению
- Минимальное количество ложных срабатываний
