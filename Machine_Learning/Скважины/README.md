# Выбор региона для бурения нефтяных скважин

## Описание проекта

Проект выполнен для компании «ГлавРосГосНефть» с целью определения оптимального региона для бурения новых нефтяных скважин. Задача заключается в анализе данных геологоразведки трех регионов и построении модели машинного обучения для предсказания объема запасов нефти в новых скважинах.

## Постановка задачи

### Основная цель
Определить регион с максимальной суммарной прибылью от добычи нефти с учетом рисков и неопределенности.

### Тип задачи
Регрессия - предсказание объема запасов нефти в скважинах (в тысячах баррелей)

### Целевая переменная
`product` - объём запасов в скважине (тыс. баррелей)

## Используемые данные

### Признаки
- **id** — уникальный идентификатор скважины
- **f0**, **f1**, **f2** — три признака точек (геологические характеристики)
- **product** — объём запасов в скважине (тыс. баррелей)

### Характеристики датасетов
- **Количество регионов**: 3
- **Размер каждого датасета**: 100,000 записей
- **Качество данных**: После предобработки остались чистые данные без пропусков

## Методология решения

### Технологический стек
- **Python** - основной язык программирования
- **Pandas** - обработка и анализа данных
- **NumPy** - численные вычисления
- **Scikit-learn** - машинное обучение
- **Matplotlib/Seaborn** - визуализация данных

### Предобработка данных
1. **Удаление дубликатов** - обнаружены и удалены неявные дубликаты по ID скважин
2. **Обработка выбросов** - применен метод IQR для удаления аномальных значений
3. **Анализ распределений** - изучены статистические характеристики признаков по регионам

### Архитектура решения
Использован **Pipeline** подход для автоматизации предобработки:
- `DecimalPointChanger` - проверка и исправление разделителей дроби
- `MissingValueHandler` - обработка пропущенных значений
- `DuplicateRemover` - удаление дубликатов
- `ImplicitDuplicatesViewer` - выявление неявных дубликатов
- `OutlierRemover` - удаление выбросов методом IQR
