# Система отбора коров для молочного хозяйства «Вольный луг»

## Описание проекта

Проект разработан для фермера, владельца молочного хозяйства «Вольный луг», который планирует расширить поголовье стада коров. Цель проекта - создать систему машинного обучения для объективного отбора коров на основе прогнозирования удоя и качества молока.

## Постановка задачи

### Основная цель
Разработать две прогнозные модели машинного обучения для отбора коров:
1. **Модель прогнозирования удоя** - предсказание годового удоя коровы
2. **Модель оценки качества молока** - расчет вероятности получения вкусного молока

### Критерии отбора
- Средний удой за год — не менее 6000 килограммов
- Молоко должно быть вкусным по критериям фермера

### Тип задач
- **Регрессия** - для прогнозирования удоя (непрерывная переменная)
- **Классификация** - для оценки вкуса молока (бинарная переменная)

## Используемые данные

### Источники данных
- **ferma_main.csv** - данные о текущем стаде фермера (616 записей после очистки)
- **ferma_dad.csv** - информация о породе отцов коров (629 записей)
- **cow_buy.csv** - данные о коровах «ЭкоФермы» для покупки (16 записей после очистки)

### Признаки

#### Параметры корма
- **ЭКЕ** - Энергетическая кормовая единица
- **Сырой протеин** - содержание в граммах
- **СПО** - Сахаро-протеиновое соотношение

#### Характеристики коровы
- **Порода** - порода коровы
- **Возраст** - бинарный признак (менее_2_лет, более_2_лет)
- **Тип пастбища** - ландшафт лугов
- **Имя папы** - кличка отца коровы
- **Порода папы быка** - порода отца

#### Характеристики молока
- **Жирность %** - содержание жиров в молоке
- **Белок %** - содержание белков в молоке
- **Вкус молока** - оценка по критериям фермера (целевая переменная)
- **Удой кг** - годовая масса молока (целевая переменная)

## Методология решения

### Технологический стек
- **Python** - основной язык программирования
- **Pandas** - обработка и анализ данных
- **Scikit-learn** - алгоритмы машинного обучения
- **Matplotlib/Seaborn** - визуализация данных

### Предобработка данных
1. **Стандартизация названий** - приведение колонок к единообразию
2. **Очистка данных** - удаление испорченных строк и выбросов
3. **Обработка типов данных** - корректное приведение типов
4. **Удаление дубликатов** - очистка от явных и неявных дубликатов
5. **Обработка выбросов** - применение метода IQR для числовых признаков

### Особенности данных
- Удалено 18 испорченных строк из основного датасета
- Устранены неявные дубликаты в категориальных признаках
- Обработаны выбросы по всем числовым характеристикам
- Стандартизированы названия пастбищ и пород

## Архитектура решения

### Подход к моделированию
- **Двухэтапное моделирование** - отдельные модели для удоя и вкуса
- **Feature Engineering** - создание новых признаков на основе существующих
- **Cross-validation** - валидация моделей для предотвращения переобучения

### Планируемые модели
1. **Линейная регрессия** - для прогнозирования удоя
2. **Логистическая регрессия** - для классификации вкуса молока
3. **Ensemble методы** - для повышения точности предсказаний

## Практическая ценность

### Для фермерского хозяйства
- **Объективный отбор** - исключение субъективных факторов при выборе коров
- **Снижение рисков** - минимизация вероятности покупки непродуктивных животных
- **Оптимизация затрат** - инвестиции только в перспективных коров
- **Планирование производства** - прогнозирование объемов молока

### Экономические преимущества
- **Увеличение рентабельности** - отбор коров с гарантированным удоем ≥6000 кг/год
- **Контроль качества** - обеспечение высокого качества молочной продукции
- **Масштабируемость** - возможность применения для больших поголовий
- **Автоматизация процесса** - сокращение времени на принятие решений

### Технические преимущества
- **Воспроизводимость** - стандартизированный процесс оценки
- **Адаптивность** - возможность обновления моделей с новыми данными
- **Интерпретируемость** - понимание факторов, влияющих на продуктивность
- **Интеграция** - возможность внедрения в существующие системы управления

## Планируемые результаты

### Модель прогнозирования удоя
- Точность предсказания удоя с погрешностью не более 10%
- Выявление ключевых факторов, влияющих на продуктивность
- Ранжирование коров по потенциальной продуктивности

### Модель оценки качества молока
- Классификация вкуса молока с точностью не менее 85%
- Определение характеристик, влияющих на качество молока
- Рекомендации по улучшению качества продукции

## Внедрение и использование

### Процесс отбора коров
1. Сбор данных о потенциальных коровах для покупки
2. Применение моделей для прогнозирования удоя и качества молока
3. Ранжирование кандидатов по комплексной оценке
4. Принятие решения о покупке на основе объективных критериев

### Мониторинг и обновление
- Регулярное обновление моделей с новыми данными
- Мониторинг точности предсказаний
- Корректировка критериев отбора при необходимости

## Заключение

Проект представляет собой комплексное решение для автоматизации процесса отбора коров в молочном хозяйстве. Использование методов машинного обучения позволяет фермеру принимать обоснованные решения о покупке животных, основываясь на объективных данных и прогнозах продуктивности.

Система обеспечивает:
- Повышение эффективности молочного производства
- Снижение финансовых рисков при расширении поголовья
- Улучшение качества молочной продукции
- Оптимизацию ресурсов фермерского хозяйства

Данный подход может быть масштабирован и адаптирован для других молочных хозяйств, что делает его ценным инструментом для развития отрасли.
