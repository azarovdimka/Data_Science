# Папка для сохранения обученных моделей

Здесь хранятся:
- heart_attack_model.cbm - оптимизированная модель CatBoost (использует 3 важных признака)

## Важные признаки модели:
1. **Systolic blood pressure** (41.86%) - систолическое артериальное давление
2. **lifestyle_score** (31.04%) - комбинированный показатель образа жизни
3. **bmi_exercise1** (27.09%) - взаимодействие ИМТ и физической активности

## Создание признаков:
- `lifestyle_score` = среднее от (курение, физ.активность, диета, сон)
- `bmi_exercise1` = BMI × Exercise Hours Per Week
- `Systolic blood pressure` = исходный признак из данных

## Оптимизация:
- Исходно: 9 признаков
- Оптимизировано: 3 признака
- Качество модели сохранено
- Скорость предсказания увеличена в 3 раза
