# Оценка риска сердечного приступа

## **Описание**
Проект для предсказания риска сердечного приступа на основе медицинских данных пациентов.

## **Структура проекта**
- `Сердечные приступы.ipynb` - основной ноутбук с EDA и обучением модели
- [другие файлы]

## **Логика исследования**
1. Загрузили и знакомились с данными в общем;
2. Определили первичные замечания по качеству предоставленных данных;
3. Создали pipleline для предобраюотки данных;
4. Классифицировали признаки на бинарные, дискретные, непрерывные для последующей визуализации и анализа распределений признаков.


Вся EDA (предобработка данных) производится в пайплайне EDAPreprocessor на классах.

## **EDAPreprocessor** - пайплайн для предобработки данных: его составляющие - классы ООП:

### **class DecimalPointChanger**:
Класс для замены разделителя дроби в строковых столбцах

**Параметры:**
- `columns`: Принимает список столбцов, в которых искать некорректный разделитель дроби;


### **class MissingValueHandler**:
Обрабатывает пропущенные значения в данных.

**Стратегии:**
- `drop`: Удаляет строки с пропусками (если < 10% данных)
- `mean`: Заполняет средним значением
- `median`: Заполняет медианой
- `mode`: Заполняет модой

### **class OutlierRemover**:
Удаляет выбросы методом IQR (только для train выборки). Значение межквартильного размаха по умолчанию 1.5, в процессе предобработки мы взяли значение шире =2.

**Параметры:**
- `columns`: Принимает список столбцов, в которых искать выбросы;

**Важно:** В медицинских данных экстремальные значения могут быть клинически значимыми! Поэтому добавлена возможность передать список столбцов, в которых допускается поиск и удаление выбросов, например в поле `"Возраст"`.

### **class ImplicitDuplicatesViewer**:
Выводит уникальные значения каждого столбца в виде списка для визуального определения неявных дубликатов.

**Параметры:**
- `columns`: Принимает список столбцов, в которых неявные дубликаты. По умолчанию ищет во всех столбцах.

### **class DuplicateRemover**:
Класс для удаления явных дубликатов.


Заупскается EDAPreprocessop pipeline при помощи функции run_preprocessor.
