
# ❤️ **Определение риска сердечного приступа**

Веб-приложение располагается по адресу http://vds.spb.su:8001

API для предсказания риска сердечного приступа на основе медицинских показателей пациентов с использованием машинного обучения (CatBoost).

**Метрики качества модели:**
* **Recall:** 0.99
* **F1-score:** 0.52
* Порог регулируется, предпочтение отдано наибольшему Recall

**Преимущества модели:**
* Превосходный Recall (99%) - критически важно в медицине, пропуск больного может стоить жизни;
* Результат значительно превышает медицинские стандарты (обычно 85-90%);
* Отсутствие переобучения - разница Train-Test всего 0.32%;
* Модель готова к внедрению в клиническую практику как инструмент скрининга;
* Использует только 3 наиболее важных признака для быстрого предсказания;
* Оптимизированная архитектура CatBoost с консервативными параметрами предотвращает переобучение.

**Используемые технологии:**
* CatBoost - градиентный бустинг для медицинских данных
* FastAPI - REST API для интеграции с медицинскими системами
* Feature Engineering - создание информативных признаков
* Автоматический подбор порога на validation выборке
