# Персонализация предложений для интернет-магазина "В один клик"

## Описание проекта

Проект направлен на разработку системы персонализированных предложений для интернет-магазина "В один клик" с целью удержания активности постоянных клиентов. Компания столкнулась со снижением покупательской активности, и привлечение новых клиентов стало менее эффективным.

## Постановка задачи

### Основная цель
Удерживать активность постоянных клиентов с помощью персональных предложений.

### Тип задачи
Классификация - предсказание уровня покупательской активности клиентов

### Целевая переменная
`Покупательская активность` - уровень активности клиента:
- "Снизилась" - клиент стал покупать меньше товаров
- "Прежний уровень" - активность осталась на том же уровне

## Используемые данные

### Файлы данных
1. **market_file.csv** - данные о поведении покупателя на сайте, коммуникациях и продуктовом поведении
2. **market_money.csv** - данные о покупательском поведении клиента (суммы трат)
3. **market_time.csv** - временные данные о поведении клиентов
4. **money.csv** - данные о прибыльности клиентов

### Группы признаков
- **Коммуникация с клиентом**: тип сервиса, разрешения на сообщения
- **Продуктовое поведение**: популярные категории, акционные покупки
- **Покупательское поведение**: суммы трат, частота покупок
- **Поведение на сайте**: количество страниц за визит, время на сайте, ошибки сервиса
