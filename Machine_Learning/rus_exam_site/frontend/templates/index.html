О системе оценки

Машинное обучение

Система использует алгоритмы машинного обучения для анализа транскрипций ответов и автоматической оценки на основе критериев экзамена.

Высокая точность

Модель обучена на большом объеме данных и показывает высокую точность в оценке ответов по критериям экзамена по русскому языку.

Критерии оценки

  • Выполнение коммуникативной задачи
  • Использование полных предложений
  • Акцент не влияет на оценку
  • Единичные ошибки не критичны
